  • Codeumfang: 1.568 Klassen, 2.932 gesamt (Groovy-Closures eingerechnet)
  • God-Objects: 11 Monster-Klassen, Spitze ShoppingCart mit 388 Methoden

  • Kein DI-Framework: Manuelle Objektverdrahtung erschwert Testbarkeit
  • Heterogene Sprachen: Java + Groovy + MiniLang-XML-DSL (deprecated)
